<p><i>World Clock</i> is a Flutter app that displays the current time and weather for various cities. The app uses the
    Material You theme to provide a modern and unique user experience.</p>
<p><br><b>Features:</b></p>
<ul>
    <li>Displays the current time for selected cities (>180)</li>
    <li>Shows the current weather (temperature, weather conditions) for the selected city</li>
    <li>Dynamic Material You theme, adapts to the color scheme of the system</li>
    <li>Highly customizable settings</li>
</ul>
